Scripting example that creates y-axis markers and modifys the datapoint colors.
BIRT Report Design that shows how to create global variables in a table script event and then use those variables to dynamically create chart markers.  The values of the markers are used as a compasison to determine each data points color.

This example was created with BIRT 2.2.0
